# valuation

评估标的当前估值水平、历史分位及相对同行位置，只读、不构成投资建议。

## 工具

- `get_valuation`：输入 `symbol`（必填），可选 `peers`（数组，用于相对估值）

## 工作流

1. 调用 `get_valuation` 获取 PE、PB、PS 及历史分位等指标。
2. 若用户提到竞争对手或同行，将 `peers` 传入以获取相对估值对比。
3. 若返回 `error` 或无有效数据，说明原因，不得估算具体倍数。
4. 有数据时解读：
   - 当前 PE/PB/PS 绝对水平
   - 历史分位（偏高/偏低/中性）
   - 相对 `peers` 的溢价或折价（若提供）
5. 区分「工具给出的数字」与「定性判断」，避免把推断写成事实。

## 约束

- 仅使用 `get_valuation`；禁止 exec 脚本路径。
- 禁止下单、禁止修改交易开关。

## 示例问法

- 「600519 现在估值贵不贵？」
- 「宁德时代相对比亚迪的 PE 分位」
