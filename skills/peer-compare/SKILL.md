# peer-compare

对 2 只及以上标的做财务横向对比，突出相对优劣与差异。

## 工具

- `peer_compare`：输入 `symbols`（数组，必填），可选 `years`（默认 5）

## 工作流

1. 从用户问题提取对比标的；若仅给出行业或简称，先确认代码再调用工具。
2. `symbols` 支持列表形式，如 `["600519.SH", "000858.SZ"]`；用户以逗号分隔时拆成数组。
3. 调用 `peer_compare` 获取同行指标表。
4. 若返回 `missing` 或 `error`，列出缺失标的，不对缺失项编造数据。
5. 对比维度建议：ROE、毛利率、净利率、营收增速、负债率、现金流质量。
6. 用表格或分点总结「谁更强、差在哪」，标注数据来源。

## 约束

- 仅使用 `peer_compare`；禁止 exec 脚本路径。
- 禁止下单、禁止修改风控。

## 示例问法

- 「茅台和五粮液财务对比」
- 「宁德时代、比亚迪、亿纬锂能同行比较」
