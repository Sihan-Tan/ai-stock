策略管理

三通道注册，统一进 StrategyRegistry：① Python 文件装饰器注册 ② YAML 规则 ③ Agent 对话生成草稿（确认后晋级）。生命周期：草稿 → 研究 → 模拟 → 实盘（需 ARM）。

① Python 注册

在 strategies/*.py 使用 @register_strategy 注册 on_bar 或 bt.Strategy；启动扫描入库。

② YAML 规则

条件 + TA-Lib 指标 + 动作；可文件与 UI 双编辑，编译为可回测/可运行策略。

③ Agent 对话

投研对话（nanobot）调用 propose_strategy → 草稿；人工确认 promote 后才能模拟/实盘。
